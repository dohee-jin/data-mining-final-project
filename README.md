## 1. 과제 개요
### 과제명: KBO 팀 성적 데이터를 활용한 포스트시즌 진출 가능성 예측 및 선수 기록 기반 골든글러브 예측

### 분석 목적
본 과제는 KBO 공식 홈페이지에서 제공하는 팀 및 선수 기록 데이터를 활용하여 팀의 포스트시즌 진출 가능성과 선수의 골든글러브 수상 가능성을 예측하는 것을 목적으로 한다. KBO 홈페이지에서는 팀 순위, 선수 순위, 타자 및 투수 기록 등을 제공하고 있어 데이터 분석에 활용하기 적합하다.

포스트시즌 진출 예측에서는 팀의 타격, 투수, 수비, 주루 기록을 활용하여 정규시즌 성적과 포스트시즌 진출 여부 사이의 관계를 분석한다. KBO는 정규리그와 퓨쳐스리그로 운영되지만 본 과제에서는 각 구단이 144경기를 치르는 정규리그 데이터만 활용한다.

골든글러브 예측에서는 선수별 타자 및 투수 기록을 활용하여 어떤 기록 지표가 수상 가능성과 관련이 있는지 분석한다. 단, 골든글러브는 단순 기록뿐 아니라 투표, 인지도, 팀 성적 등 정량화하기 어려운 요소도 영향을 줄 수 있으므로 본 과제에서는 “정확한 수상자 예측”보다는 기록 기반 후보 예측에 초점을 둔다.

## 2. 데이터 수집 및 전처리

### 수집 데이터
본 과제에서는 KBO에서 제공하는 기록 데이터를 중심으로 수집한다.

### 1) 선수기록
| 구분    | 수집 항목 예시                                    |
| ----- | ------------------------------------------- |
| 타자 기록 | 타율, 경기 수, 타석, 안타, 홈런, 타점, 득점, 출루율, 장타율, OPS |
| 투수 기록 | 평균자책점, 승, 패, 세이브, 홀드, 이닝, 탈삼진, WHIP, ERA         |
| 주루 기록 | 도루, 도루 성공률, 주루 관련 기록                        |
| 수비 기록 | 실책, 수비율, 포지션별 기록                            |

### 2) 팀 기록
| 구분      | 수집 항목 예시                  |
| ------- | ------------------------- |
| 팀 타자 기록 | 팀 타율, 득점, 안타, 홈런, 타점, OPS |
| 팀 투수 기록 | 팀 평균자책점, 실점, 탈삼진, WHIP, ERA    |
| 팀 주루 기록 | 도루, 도루 성공률                |
| 팀 수비 기록 | 실책, 수비율                   |

### 3) 목표 변수
| 분석 주제       | 목표 변수                        |
| ----------- | ---------------------------- |
| 포스트시즌 진출 예측 | 진출 = 1, 미진출 = 0              |
| 골든글러브 예측    | 수상 또는 후보 가능성 높음 = 1, 그 외 = 0 |

포스트시즌 진출 여부는 정규시즌 순위를 기준으로 구분한다. 일반적으로 포스트시즌 진출 팀과 미진출 팀을 나누어 이진 분류 문제로 구성할 수 있다.

### 전처리 과정
1. 필요한 칼럼 선택   
분석에 필요한 팀 기록, 선수 기록 칼럼만 선택한다. 예를 들어 포스트 시즌 진출 예측의 경우 팀 타율, 득점, 홈런, 평균자책점, 실책 등의 변수를 사용한다.

2. 결측치 처리   
기록이 비어있는 값이 있는지 확인하고 필요에 따라 제거하거나 평균값 등 으로 대체한다.

3. 데이터 형식 변환   
타율, 평균자책점, OPS 등 숫자형으로 분석해야 하는 값들이 문자열로 지정되어 있다면 숫자형 데이터로 변환한다.

4. 라벨 생성   
포스트 시즌 진출여부는 `진출 = 1`, `미진출 = 0`으로 변환한다. 골든글러브 예측도 수상 여부 또는 후보 여부를 기준으로 `1`과 `0`으로 변환한다.

5. 정규화 또는 표준화   
로지스틱 회귀나 퍼셉트론처럼 숫자 크기에 영향을 받을 수 있는 모델을 사용할 경우 변수들의 단위를 맞추기 위해 정규화 또는 표준화를 적용한다.

6. 학습 데이터와 테스트 데이터로 분리   
전체 데이터를 학습용 데이터와 테스트용 데이터로 나누어 모델의 예측 성능을 확인한다.

## 3. 적용 데이터 마이닝 기법
### 1) 분류    
본 과제의 핵심 분석 방법은 분류이다. 포스트시즌 진출 여부와 골든글러브 후보 여부는 모두 `1`, `0`으로 나눌 수 있으므로 이진 분류 문제로 볼 수 있다.   

예를 들어 포스트시즌 진출 예측은 다음과 같이 구성할 수 있다.
| 팀  | 기록 데이터            | 결과  |
| -- | ----------------- | --- |
| A팀 | 팀 타율, 홈런, 평균자책점 등 | 진출  |
| B팀 | 팀 타율, 홈런, 평균자책점 등 | 미진출 |


### 2) 로지스틱 회귀
로지스틱 회귀는 결과가 `0`과 `1`로 나뉘는 이진 분류 문제에 적합한 모델이다.   

본 과제에서 로지스틱 회귀를 활용하여 팀의 포스트시즌 진출 결과를 확률 형태로 예측할 수 있다. 예를 들어 특정 팀의 포스트시즌 진출 결과가 `0.78`로 나오면, 해당 팀은 포스트시즌 진출 가능성이 높은 팀으로 해석할 수 있다.   

골든글러브 예측에서도 선수별 수상 가능성을 확률 형태로 계산하여 확률이 높은 선수를 후보로 제시할 수 있다.

### 3) 의사결정트리
의사결정트리는 데이터를 여러 조건으로 나누어 결과를 예측하는 모델이다. 이 모델은 결과를 해석하기 쉽다는 장점이 있다.   

예를 들어 포스트시즌 진출 예측에서는 다음과 같은 방식으로 설명할 수 있다 
```
팀 승률이 높은가?
→ 팀 평균자책점이 낮은가?
→ 득점이 많은가?
→ 포스트시즌 진출 가능성 높음
```   

골든글러브 예측에서는 다음과 같은 해석이 가능하다.
```
OPS가 높은가?
→ 경기 수가 충분한가?
→ 홈런 또는 타점이 높은가?
→ 골든글러브 후보 가능성 높음
```
따라서 의사결정트리는 단순히 예측 결과를 내는 것 뿐만 아니라 어떤 기록이 예측에 중요한 영향을 주는지 설명하는데 활용할 수 있다.

### 4) 퍼셉트론
퍼셉트론은 기본적인 선형 분류 모델이다. 본 과제에서는 포스트시즌 진출 여부와 골든글러브 후보 여부를 구분하는 비교 모델로 활용할 수 있다.

다만 퍼셉트론은 복잡한 데이터 관계를 설명하는 데 한계가 있을 수 있으므로, 로지스틱 회귀와 의사결정트리의 결과와 비교하는 보조 모델로 사용한다.

### 5) 군집화 
군집화는 정답 라벨 없이 비슷한 특징을 가진 데이터를 그룹으로 나누는 기법이다.

본 과제에서는 군집화를 보조 분석으로 활용할 수 있다. 예를 들어 팀 데이터를 기준으로 다음과 같은 군집을 만들 수 있다.   

| 군집    | 특징 예시                 |
| ----- | --------------------- |
| 공격형 팀 | 득점, 홈런, 팀 OPS가 높은 팀   |
| 투수형 팀 | 평균자책점, WHIP, 실점이 낮은 팀 |
| 균형형 팀 | 타격과 투수 지표가 모두 안정적인 팀  |  


선수 데이터에도 군집화를 적용하여 다음과 같이 나눌 수 있다.
| 군집      | 특징 예시                  |
| ------- | ---------------------- |
| 장타형 타자  | 홈런, 장타율, 타점이 높은 선수     |
| 출루형 타자  | 출루율, 타율, 볼넷 비율이 높은 선수  |
| 주루형 선수  | 도루, 도루 성공률이 높은 선수      |
| 에이스형 투수 | 평균자책점, 이닝, 탈삼진이 우수한 투수 |
| 불펜형 투수  | 세이브, 홀드 기록이 높은 투수      |

군집화 결과는 분류 모델의 예측 결과를 해석하는 데 보조적으로 활용할 수 있다.

## 4. 분석 프로세스
본 과제는 다음과 같은 순서로 진행한다.

### 1단계: 데이터 수집
KBO 공식 홈페이지에서 팀 기록과 선수 기록 데이터를 수집한다. 팀 기록은 포스트시즌 진출 예측에 사용하고 선수 기록은 골든글러브 후보 예측에 사용한다. 

### 2단계: 데이터 전처리
수집한 데이터에서 필요한 칼럼을 선택하고 결측치와 이상치를 확인한다. 이후 숫자형 데이터 변환, 라벨 생성, 정규화 또는 표준화 과정을 수행한다.

### 3단계: 탐색적 데이터 분석
포스트시즌 진출 팀과 미진출 팀의 기록 차이를 비교한다. 예를 들어 진출 팀의 평균 득점, 평균자책점, 홈런 수 등이 미진출 팀과 어떤 차이가 있는지 확인한다.

골든글러브 분석에서는 수상자 또는 후보 선수들의 타율, 홈런, 타점, OPS 등이 일반 선수들과 어떤 차이가 있는지 비교한다.

### 4단계: 모델 적용
수업에서 학습한 로지스틱 회귀, 의사결정트리, 퍼셉트론 모델을 적용한다. 포스트시즌 진출 여부와 골든글러브 후보 여부를 각각 예측하고, 모델별 예측 성능을 비교한다.

### 5단계: 군집화 분석
팀 또는 선수 데이터를 대상으로 군집화를 수행하여 비슷한 특징을 가진 그룹을 찾는다. 군집화 결과를 활용해 특정 유형의 팀이나 선수가 예측 결과와 어떤 관련이 있는지 분석한다.

### 6단계: 결과 해석 및 시각화
모델의 정확도, 혼동행렬, 주요 변수 등을 확인한다. 의사결정트리의 경우 어떤 조건이 예측에 중요하게 작용했는지 해석한다. 결과는 표와 그래프로 시각화하여 정리한다.

### 7단계: 결론 도출
분석 결과를 바탕으로 포스트시즌 진출에 영향을 주는 주요 팀 기록과 골든글러브 후보 선정에 영향을 주는 주요 선수 기록을 정리한다.

## 5. 기대 효과 및 한계점
### 기대 효과
첫째, KBO 공식 기록 데이터를 활용하여 실제 스포츠 데이터를 기반으로 데이터 마이닝 기법을 적용해볼 수 있다.

둘째, 수업에서 배운 분류, 로지스틱 회귀, 의사결정트리, 퍼셉트론, 군집화 개념을 실제 데이터 분석 과정에 연결할 수 있다.

셋째, 단순히 순위나 기록을 확인하는 것을 넘어서, 어떤 지표가 포스트시즌 진출이나 골든글러브 후보 가능성과 관련이 있는지 분석할 수 있다.

넷째, 의사결정트리와 군집화 결과를 활용하면 예측 결과를 비교적 쉽게 해석할 수 있어 발표와 보고서 작성에 유리하다.

### 한계점
첫째, 포스트시즌 진출 여부는 선수 부상, 외국인 선수 교체, 시즌 후반 경기력, 팀 분위기 등 기록 데이터만으로 설명하기 어려운 요인의 영향을 받을 수 있다.

둘째, 골든글러브는 공식 기록뿐 아니라 기자단 투표, 선수 인지도, 팀 성적, 시즌 임팩트 등이 영향을 줄 수 있으므로 기록만으로 정확한 수상자를 예측하는 데 한계가 있다.

셋째, 골든글러브 수상자는 전체 선수 수에 비해 매우 적기 때문에 데이터 불균형 문제가 발생할 수 있다. 따라서 본 과제에서는 정확한 수상자 예측보다는 기록 기반 후보 예측에 초점을 둔다.

넷째, KBO 시즌 수가 제한적이기 때문에 충분한 학습 데이터를 확보하기 어려울 수 있다. 이를 보완하기 위해 여러 시즌의 데이터를 함께 수집하고, 모델 성능보다는 분석 과정과 해석에 중점을 둔다.

---

본 과제는 KBO 팀 및 선수 기록 데이터를 활용하여 포스트시즌 진출 여부와 골든글러브 후보 가능성을 예측하는 프로젝트이다. 수업에서 학습한 분류, 로지스틱 회귀, 의사결정트리, 퍼셉트론, 군집화 기법을 적용하여 실제 스포츠 데이터에서 의미 있는 패턴을 찾고, 예측 결과와 한계점을 함께 분석하고자 한다.
